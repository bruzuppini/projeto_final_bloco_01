# Projeto Final do Bloco 1 - Loja de Jogos Indie (Bootcamp Generation Brazil)

Projeto backend de uma loja de jogos indie, desenvolvido em TypeScript e Node.js. A aplicação gerencia catálogo de jogos, usuários e pedidos, aplicando programação orientada a objetos e boas práticas de estruturação de código. Este projeto demonstra habilidades em tipagem estática, organização de classes e funções, e lógica aplicada em cenários de e-commerce de jogos digitais.
